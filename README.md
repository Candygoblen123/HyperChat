## Building from Source
Clone the repository recursively:
```bash
git clone https://github.com/LiveTL/HyperChat --recursive
```
Open the repository and npm install:
```bash
cd hyperchat
yarn
```
Build or serve the extension:
```bash
yarn build # prod
yarn start # dev
yarn test  # jest
```
